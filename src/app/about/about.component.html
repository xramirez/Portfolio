My Projects

Most of the projects that I've worked on over the years, you can find on my GitHub Page! However, there are a few I am particularly proud of creating, and I wanted to showcase them here!
My Dashboard
  • I Created a Dashboard recently as a project under TalentPath! This dashboard is built out of HTML5/CSS and JavaScript, and utilizes multiple APIs to gather information such as the weather, when the nearest MCU movie is, and a database API that allowed me to create a dynamic to-do list from scratch whose information is saved entirely online!
My Trivia Game
  • I created a Trivia Game under TalentPath as well! This trivia game is built as a project in Angular with Typescript, and is powered by the OpenTriviaDB API, and its information is saved with the M3O Database API! After a game is completed, it stores a user's statistics and displays them in relation to other people who have previously played the game.
Stocks.io
  • One of my biggest projects, that was only created over the course of a week and a half! Created with Chase Faggard, we created a Stock Display Dashboard in Angular! The dashboard communicates through a Socket.io server, which creates randomized fake stock data and feeds it to the dashboard, which displays the data in candlestick charts.
